# 🎮 Rock Paper Scissors (Python)

A simple and interactive **Rock Paper Scissors game** built using Python. This command-line application allows users to play against the computer with randomized choices, input validation, and replay functionality.

---

## 🚀 Features

* 🎯 User vs Computer gameplay
* 🔁 Replay option to play multiple rounds
* ⚡ Fast and lightweight CLI-based game
* 🧠 Randomized computer moves using `random` module
* 🛡️ Input validation to prevent crashes
* 🎨 ASCII art for enhanced visual experience

---

## 🛠️ Tech Stack

* **Language:** Python
* **Concepts Used:**

  * Conditional Statements
  * Functions
  * Loops
  * Lists
  * Exception Handling

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/codes-by-aditi/rock-paper-scissors-python.git
   ```

2. Navigate to the project folder:

   ```bash
   cd rock-paper-scissors-python
   ```

3. Run the game:

   ```bash
   python rock_paper_scissors.py
   ```

* Rock beats Scissors
